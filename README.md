# react-rails redux sample
this is a simple example application with CRUD api using react-rails and redux
* rails 5.0.0.1
* redux 3.0.4
* react-rails 1.10.0
* react 15.4.2

## Installation

```
git clone https://github.com/jekaantipin/react-rails-redux-sample.git
cd react-rails-redux-sample
bundle install
npm install
```
